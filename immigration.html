데이터 연동 준비중

공공데이터포털(data.go.kr) 서비스키 발급이 필요합니다

법무부 출입국·외국인정책본부의 체류외국인 통계는 data.go.kr에서 "법무부" + 원하는 통계명(예: 국적별 체류외국인 현황)으로 검색 → 활용신청 → 서비스키 발급 후 REST(JSON) 호출로 연동합니다 (PROJECT_SPEC.md 섹션 1-1). hikorea.go.kr 직접 크롤링은 사용하지 않습니다. 서비스키가 준비되면 scripts/fetch_immigration.py가 GitHub Actions에서 이 키(Secrets)로 데이터를 받아와 data/immigration_stats.json을 생성하고, 아래 카드/차트가 자동으로 채워집니다.
