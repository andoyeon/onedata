# 한국은행 기준금리 동향 (2026년 7월 기준)

## 현재 기준금리

- **2.50%** (2025년 5월 29일 인하 이후 유지)
- 2026년 5월 금융통화위원회 회의에서 8회 연속 동결

## 최근 변동 이력

| 결정일 | 변경 내용 | 기준금리 |
|---|---|---|
| 2021.08 ~ 2023.01 | 인상 사이클 (총 12회 인상) | 0.50% → 3.50% |
| 2023.02 ~ 2024.08 | 3.50%에서 장기 동결 (13회 연속) | 3.50% |
| 2024.10.11 | 인하 (첫 인하) | 3.50% → 3.25% |
| 2024.11.28 | 인하 | 3.25% → 3.00% |
| 2025.02.25 | 인하 | 3.00% → 2.75% |
| 2025.04.17 | 동결 | 2.75% |
| 2025.05.29 | 인하 | 2.75% → 2.50% |
| 2025.05 ~ 2026.05 | 2.50%에서 장기 동결 (8회 연속) | 2.50% |

## 배경 및 전망

- 2021~2023년 인상 사이클은 코로나19 이후 물가 급등에 대응하기 위한 조치였으며, 기준금리를 0.50%에서 3.50%까지 총 3.00%p 인상.
- 2024년 10월부터 시작된 인하 사이클은 물가 안정과 내수 부진 대응 목적으로, 2025년 5월까지 총 4차례(1.00%p) 인하하여 2.50%에 도달.
- 2026년 들어 소비자물가 상승률이 3.2% 수준으로 목표(2.0%)를 상회하고 있어, 한국은행은 추가 인하보다는 동결 기조를 유지하는 중.
- 시장 전문가 다수는 2026년 중 기준금리가 물가·성장·금융안정 상황을 종합적으로 고려하며 2.50% 수준에서 유지될 것으로 전망.
